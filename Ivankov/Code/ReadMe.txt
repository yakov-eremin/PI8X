Для запуска приложения достаточно запустить исполняемый файл ...Virtual museum\Build\Virtual museum.exe
Для сборки приложения необходимо:
1) Скачать приложение Unity Hub
2) В Unity Hub скачать игровой движок Unity 2019.<свежая версия> (На данный момент 2019.4.29f1)
3) Скачать проект по ссылке в файле LinkToProject.
4) Подключить проект в Unity Hub
5) Запустить проект в Unity Hub

01. 	Взаимодействие с экспонатом
Действующее лицо: пользователь
Предусловие:	куплен хотя бы один гид определенного зала, где находится экспонат;
		персонаж должен стоять рядом с экспонатом, курсор указывает на экспонат;
		при указании курсора на экспонат всплывает подсказка об открытии гида
Основной сценарий:
•	Нажать клавишу F на клавиатуре, чтобы открыть гид
Постусловие: открытие нового всплывающего окна музейного гида

02. 	Просмотр текстового гида экспоната
Действующее лицо: пользователь
Предусловие:	открыто всплывающее окно музейного гида определенного экспоната
Основной сценарий:
•	Нажать кнопку "Текстовый гид". Открывается дополнительное окно описания экспоната
	Альтернативный сценарий:
	•	Нажать клавишу "Ссылка на создателя". Описание экспоната расширяется ссылкой о создателе. Повторное нажатие на кнопку скрывает ссылку из описания.
	•	Нажать клавишу "Ссылка на экспонат". Описание экспоната расширяется ссылкой об экспонате. Повторное нажатие на кнопку скрывает ссылку из описания.
•	Повторное нажатие на кнопку "Текстовый гид" скроет дополнительное окно описания экспоната
Постусловие: обновленное всплывающее окно музейного гида определенного экспоната

03. 	Просмотр аудиогида экспоната
Действующее лицо: пользователь
Предусловие:	открыто всплывающее окно музейного гида определенного экспоната
Основной сценарий:
•	Нажать кнопку "Аудиогид". Открывается дополнительное окно управление аудиогидом
•	Нажать кнопку со значком крестика. Обновляется статус аудиогида на "Остановлен"
•	Нажать кнопку со значком воспроизведения. Обновляется статус аудиогида на "Воспроизводит"
•	Нажать кнопку со значком паузы. Обновляется статус аудиогида на "На паузе"
•	Повторное нажатие на кнопку "Аудиогид" скроет дополнительное окно управление аудиогидом
Постусловие: обновленное всплывающее окно музейного гида определенного экспоната

04. 	Закрытие музейного гида
Действующее лицо: пользователь
Предусловие:	открыто всплывающее окно музейного гида определенного экспоната
Основной сценарий:
•	Нажать клавишу "Закрыть гид"
Постусловие: закрытие всплывающего окна музейного гида; возможность передвижения персонажа при помощи кнопок 'wasd'

05. 	Переход в кассу
Действующее лицо: пользователь
Предусловие:	персонаж находится в помещении музея
Основной сценарий:
•	Управляя персонажем при помощи кнопок на клавиатуре 'wasd' подойти к терминалу-кассе, который находится перед входами в залы
Постусловие: открытие интерфейса кассы

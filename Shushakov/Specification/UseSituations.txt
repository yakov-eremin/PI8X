1)	Открытие изображения. 
Действующее лицо: пользователя приложения. 
Предусловие: пользователь находится в главном окне приложения. 
Основной сценарий: 
•	Нажать на кнопку «Загрузить изображение». Откроется диалоговое окно для выбора файла.
•	В диалоговом меню выбрать изображение для открытия. Нажать «Открыть».
Постусловие: в главном окне приложения появится выбранное изображение.

2)	Выбор области для коррекции цветов изображения.
Действующее лицо: пользователь приложения.
Предусловие: пользователь открыл изображение для коррекции. Выбрана предыдущая область (либо вся область) изображения.
Основной сценарий:
•	Сделать нажатия ЛКМ в области отображения изображение и не отпуская ЛКМ растянуть область выделения.
•	Отпустить ЛКМ. Будет выбрана новая прямоугольная область изображения для коррекции цветов. Повторное нажатие ЛКМ над изображением выберет всю область изображения.
•	На гистограмме справа от изображения отобразится гистограмма яркости выделенной области.
Постусловие: Пользователем выбрана новая область для коррекции изображения.

3.	Выбор режима Коррекции цветов изображения
Действующее лицо: пользователь приложения.
Предусловие: пользователь открыл изображение. Выбрал область для коррекции цветов или оставил ее без изменений.
Основной сценарий:
•	Нажать на кнопку «Коррекция изображения».
•	Выбрать режим коррекции изображения в выпадающем меню. Нажать на него ЛКМ. 
•	Выбранная область изображения будет корректирована. Пользователю будет предложено внести параметры для коррекции в диалоговом окне. Пользователь выбирает настройки для корректировки. Изменения настроек применяются в режиме реального времени.
•	Нажать на кнопку «Применить» чтобы применить изменения, либо на кнопку «Отменить», чтобы отменить изменения.
Постусловие: Область для коррекции цветов изображения будет откорректирована в соответствие с выбранным режимом и введенными настройками при изменении настроек.

4.	Применение фильтров.
Действующее лицо: пользователь приложения.
Предусловие: пользователь открыл изображение. Выбрал область для коррекции цветов или оставил ее без изменений.
Основной сценарий:
•	Нажать на кнопку «Фильтры».
•	Выбрать режим применения фильтров для изображения в выпадающем меню. Нажать на него ЛКМ. 
•	Для выбранной области изображения будет применен фильтр. Пользователю будет предложено внести параметры для коррекции в диалоговом окне. Пользователь выбирает настройки для корректировки цветов. Изменения настроек применяются в режиме реального времени.
•	Нажать на кнопку «Применить» чтобы применить изменения, либо на кнопку «Отменить», чтобы отменить изменения.
Постусловие: для области изображения будет применена фильтрация в соответствие с выбранным режимом и введенными настройками при изменении настроек.

5.	Восстановление изображения.
Действующее лицо: пользователь приложения.
Предусловие: произвел изменения над открытым изображением. 
Основной сценарий:
•	Нажать на кнопку «Восстановить изображение» либо нажать Ctrl+Z.
Постусловие: измененное изображения вернется в первоначальное состояние, соответствующее тому, которое было после загрузки изображения.

6.	Сохранение изменений.
Действующее лицо: пользователь приложения.
Предусловие: произвел изменения над открытым изображением. 
Основной сценарий:
•	Нажать на кнопку «Сохранить».
•	В диалоговом меню выбрать путь и название файла для сохранения. Нажать «Сохранить».
Постусловие: измененное изображения сохранится на диск

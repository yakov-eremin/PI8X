Подробное описание

Базовый класс модели представления. Реализует базовую функциоанальность модели представления для осуществления работы паттерна MVVM.

Является расширением разметки, поэтому объекты ViewModel можно использовать в xaml разметке.
Базово реализует интерфейс INotifyPropertyChanged. Реализацию не следует переопределять.

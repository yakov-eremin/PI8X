◆ FullTimeReserve

int AstLab3.Models.Schedules.Work.FullTimeReserve
get

Полный резерв времени работы

Полный резерв времени работы вычисляется по формуле \(x_l - y_e - length\), где \(x_l\) - поздний срок сверешния конечного события, \(y_e\) - ранний срок сверешния начального события, \(length\) - продолжительность работы.

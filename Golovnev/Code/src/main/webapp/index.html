Курсовая работа. Справочник контрагентов

Работу выполнил Головнев Н. А.

В данной курсовой работе была реализована модель данных и сервисы для работы со «Справочником контрагентов».

Справочник контрагентов содержит следующие поля + (уникальный идентификатор в качестве первичного ключа):
1. Наименование
2. ИНН
3. КПП
4. Номер счёта
5. БИК банка


Пользовательский сценарий 1. Сделать запись видео вокруг курсора.
Действующее лицо: Пользователь

Предусловие: Пользователь находится в главном окне; Пользователь не начал запись
Основной сценарий:
1) Пользователь выбрал режим записи вокруг курсора, нажав на кнопку курсора.
2) Пользователь настроил размер записываемой области вокруг курсора
	Алтернативный сценарий: Пользователь на настроил размер записываемой области вокруг курсора. Приминились настройки по умолчанию.
3) Пользователь настроил режимы записи звука
	Альтернативный сценарий: Пользователь не стал настраивать режимы записи звука. Приминились настройки по умолчанию(микрофон и динамик выключены)
4) Пользователь нажал на кнопку начать запись в левом верхнем углу приложения.

5) Пользователь может управлять режимами записи звука.

6) Пользователь нажал на кнопку остановить запись в левом верхнем углу приложения. В правом нижнем углу экрана появляется всплывающее окно Notification.

7) Пользователь нажимает на кнопку Close. Окно Notification закрывается. 
	Альтернативный сценарий: 7) Пользователь нажимает на кнопку Delete. Всплывает окно подтверждения удаления записанного видео.
				 8) Пользователь нажимает на кнопку Да. Видео удаляется.
					Альтернативный сценарий: 8) Пользователь нажимает на кнопку Нет. Отмена удаления.

Постусловие: Видео сохранено.
